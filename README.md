# Maal Wallet — Android APK

Latest build of the Maal Wallet mobile app (mobile money + interest-free BNPL + agent CICO).

- **Backend:** https://api.146.148.39.5.sslip.io (deployed)
- **Package:** `com.example.maal_wallet_app`
- **Version:** 1.0.0

## Install
1. Download **[Maal-Wallet.apk](./Maal-Wallet.apk?raw=true)** on your Android phone.
2. Allow installs from this source if prompted, then open the APK to install.
3. Register with your phone number; the app talks to the deployed backend.

Roles supported: customer, agent (cash-in/out), and admin-driven flows.
